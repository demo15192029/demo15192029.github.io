Description of Requirements

Our game style is cartoon funny.
Some of the characters in this link are similar to ours, just for style reference...
https://www.youtube.com/watch?v=-uwZIdcflDE

These sound effects need to make sure they can be used directly,
especially when it comes to movement and flying,
which will involve continuous flight and repeat playback in a natural way,
so you need to consider that we can use them easily.
we need you to use your imagination. Please do your best.
we want to make a beautiful work and include your name in the list of game productions.

Each character usually contains the following actions...
Coming, Ready, Moving, Flying, Fighting, Skill, Injured, Resting, laughing, Dying
 The ones marked in white are the sounds we need.
In order to accurately produce each animation, the duration
of each sound must be an integer e.g. 0:00:400  0:01:200  0:02:300

The naming method is "character name & action" (wav/mp3 320kbps)
For example: dragonfly-moving.mp3 (all lowercase)
